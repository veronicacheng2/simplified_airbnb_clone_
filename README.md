# Simplified Airbnb Clone (MERN Stack)
Frontend:Reactjs <br>
Backend:Nodejs,Expressjs <br>
Database:MongoDB <br>

## Features
1. User Registration
2. Authentication (JWT)
3. Uploading Image
4. Create Booking
5. Listing Your Place
